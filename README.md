# Fontys Electrical Engineering MO2 project
## GUI
"/Metal Detector WinForms" is a visual studio code Windows forms project  

## Backend
"/Metal Detector Arduino" is a visual studio code project which requires PlatformIO extension  
this project contains 5 different builds:  
1. nano_33_iot_BUILD  
2. nano_33_iot_SIMULATE_SENSORS  
3. nano_33_iot_SIMULATE_MEASUREDATA  
4. Native_SingleThreaded_SIMULATION_DATA  
5. Native_MultiThreaded_SIMULATION_DATA
